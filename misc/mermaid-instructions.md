# Pipeline diagram generation

The pipeline diagram is generated by mermaid, using these files:

 * `mermaid-config.json`: Mermaid configuraion
 * `pipeline-diagram.mmd`: Mermaid representation of the pipeline

 [https://mermaid.d.foundation](https://mermaid.d.foundation) can be used with the above files to generate the diagram.

 Save the file in this directory as `pipeline-diagram.svg`
