# Pipeline diagram generation

The pipeline diagram is generated by mermaid, using these files:

 * `pipeline-diagram.mmd`: Mermaid representation of the pipeline

To generate:

 * Install mermaid
   ```sh
   npm install @mermaid-js/mermaid-cli
   ```
 * Change to directory containing these instructions
 * Generate the diagram
   ```sh
   mmdc -i pipeline-diagram.mmd -o pipeline-diagram.svg
   ```
